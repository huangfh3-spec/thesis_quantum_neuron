# Environment Setup

The simulations were developed using **Python** and the **QuTiP quantum simulation framework**.

To reproduce the computational environment, create the conda environment:

```
conda env create -f environment.yml
conda activate qutip_repro
```

Main dependencies include:

- Python  
- NumPy  
- SciPy  
- Matplotlib  
- Pandas  
- QuTiP  

---

# Running the Simulation Scripts

All figure scripts are located in the `scripts/` directory.

To run a specific simulation, navigate to the repository root directory and execute:

```
python scripts/fig4-10.py
```

This will generate the corresponding figure and save it in the `figures/` directory.

Example:

```
python scripts/fig4-16a.py
```

# Notes

1. The scripts implement the **numerical simulation of the quantum neuron model** described in Chapter 4 of the thesis.

2. The model is based on solving the **Lindblad master equation** for a seven-level trapped-ion system.

3. Different scripts correspond to different neuronal behaviors, including spiking dynamics, firing patterns, and XOR classification.
